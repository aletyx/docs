Within Aletyx Enterprise Build of Kogito and Drools there is a robust and flexible framework for business automation, integrating workflows, decision management, and task orchestration. This document explores its **core components**, starting with the **Engine**, which powers process execution, followed by **Runtimes**, which define the execution environment.

We then dive into data, covering **Data-Index and Data-Audit**, essential for tracking and querying process states. The **Job Service** enables scheduled task execution, while the **User Task Subsystem** facilitates human interaction within automated workflows.

Together, these components create a flexible and efficient automation ecosystem, providing developers with the necessary tools to build resilient and adaptable business processes.
