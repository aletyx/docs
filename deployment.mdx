## Quick start

Build with **Maven**, create **Docker** images, and run containers with simple
CLI commands. For Decision Control specifically, you can start with a two-minute
Docker setup or evaluate via the AWS Marketplace sandbox before moving to a
Kubernetes multi-environment configuration for production.

## Production considerations

- **CI/CD pipelines** with Git workflows and automated testing.
- **Horizontal scaling** via Kubernetes StatefulSets and auto-scaling (HPA).
- **High availability** through load balancing, database replication, and
  multi-AZ deployment.

## Choosing a target

| Target | Setup time | Complexity | Production ready |
|---|---|---|---|
| Docker | Minutes | Low | Dev/test |
| Docker Compose | Minutes | Low | Dev/test |
| AWS Marketplace sandbox | Minutes | Low | Evaluation |
| Kubernetes | Longer | Higher | Yes |
