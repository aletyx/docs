The Aletyx Enterprise Build is a comprehensive business automation platform
built on the **Apache KIE** open source projects, combining proven reliability
with enterprise-ready features for Java teams working in process and decision
automation. The platform emphasizes modular architecture, independent scaling,
and flexible deployment across cloud, on-premises, or hybrid environments.

## Why an enterprise build

The Apache KIE projects — Drools, jBPM, and Kogito — are mature, widely adopted
open source automation engines. The Aletyx Enterprise Build hardens them for
production with long-term support, security updates, and the governance,
deployment, and integration tooling enterprises need.
