## Overview

<Note>
Deploy Aletyx Decision Control in minutes with pre-configured AMIs and automated CloudFormation templates. All infrastructure, SSL certificates, and security configuration handled automatically.
</Note>

**Available Edition:**

- **Aletyx Decision Control Sandbox** - Quick evaluation with embedded H2 database, pay-as-you-go hourly billing

## Getting Started

Choose your path based on your role:

### For DevOps/Platform Teams

**Goal:** Deploy Decision Control in your AWS environment

**→ Start here:** [Sandbox Edition](/deployment/aws-marketplace/sandbox)

You'll learn how to launch CloudFormation stacks, configure networking, and set up SSL/HTTPS.

### For Business Analysts & Decision Modelers

**Goal:** Access Decision Control application after deployment

**→ Start here:** [Sandbox Edition](/deployment/aws-marketplace/sandbox)

You'll learn how to find your application URL and log in to start creating decision models.

### For Security & Compliance Teams

**Goal:** Understand security controls and best practices

**→ Start here:** [Security Best Practices](/deployment/aws-marketplace/security)

You'll learn about IP restrictions, encryption, audit trails, and compliance features.

### For Application Developers

**Goal:** Connect to Decision Control APIs

**→ Start here:** [Sandbox Edition](/deployment/aws-marketplace/sandbox) then [Troubleshooting](/deployment/aws-marketplace/troubleshooting)

You'll learn how to SSH or use SSM Session Manager to configure integrations.

## Deployment Steps

All AWS Marketplace deployments follow these steps:

1. **Subscribe** on AWS Marketplace
2. **Configure** CloudFormation parameters
3. **Deploy** via one-click CloudFormation
4. **Access** your application (5-10 minutes)

Detailed instructions:

- [Sandbox Edition Deployment](/deployment/aws-marketplace/sandbox) - Quick evaluation setup
