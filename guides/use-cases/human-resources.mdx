## 1. Employee Onboarding Process

### Business Scenario

A technology company creating a comprehensive, streamlined onboarding experience for new hires across multiple departments.

  <img src="/images/uc-human-resources-flow-1-dark.svg" alt="Flowchart of an employee onboarding process: Job Offer Acceptance, Pre-Boarding Preparation, Document Collection and Background Check (with HR investigation when needed), IT Resource Allocation and Training Schedule, then First Day Orientation, Department Introduction and Role-Specific Training, ending with a 30-60-90 day check-in plan and performance baseline" className="hidden dark:block" />
</Frame>

### Key Process Details

| Stage                 | Description                       | Duration  | Key Actors          |
| --------------------- | --------------------------------- | --------- | ------------------- |
| Pre-Boarding          | Initial paperwork and preparation | 1-2 weeks | HR Team             |
| Document Verification | Background and legal checks       | 3-5 days  | HR Compliance       |
| Resource Allocation   | IT, workspace, and tool setup     | 2-3 days  | IT and Facilities   |
| Orientation           | Initial company introduction      | 1-2 days  | HR and Management   |
| Training              | Role-specific skill development   | 2-4 weeks | Department Managers |

## 2. Performance Review Process

### Business Scenario

A multinational corporation implementing a comprehensive, data-driven performance management system.

### Process Overview

<Frame>
  <img src="/images/uc-human-resources-flow-2.svg" alt="Flowchart of a performance management cycle: Performance Period Begins, Goal Setting, Continuous Feedback, Self-Assessment, Manager Review and Performance Evaluation, routing to a High Performer track with promotion and compensation review, standard feedback with a development plan, or a Performance Improvement Program" className="block dark:hidden" />
  <img src="/images/uc-human-resources-flow-2-dark.svg" alt="Flowchart of a performance management cycle: Performance Period Begins, Goal Setting, Continuous Feedback, Self-Assessment, Manager Review and Performance Evaluation, routing to a High Performer track with promotion and compensation review, standard feedback with a development plan, or a Performance Improvement Program" className="hidden dark:block" />
</Frame>

### Key Process Details

| Stage                | Description                          | Duration   | Key Actors           |
| -------------------- | ------------------------------------ | ---------- | -------------------- |
| Goal Setting         | Establish performance objectives     | 1-2 weeks  | Employee and Manager |
| Continuous Feedback  | Ongoing performance tracking         | Continuous | All Stakeholders     |
| Self-Assessment      | Employee performance reflection      | 1 week     | Employee             |
| Manager Review       | Comprehensive performance evaluation | 2-3 weeks  | Managers             |
| Development Planning | Improvement and growth strategies    | 1-2 weeks  | HR and Management    |

## 3. Leave Request Management

### Business Scenario

A global organization creating a transparent and efficient leave management system.

### Process Overview

<Frame>
  <img src="/images/uc-human-resources-flow-3.svg" alt="Flowchart of a leave request process: Leave Request Submission, Leave Type Verification (vacation, sick or special), Available Balance Check with medical documentation and compliance review for sick or special leave, then Manager Approval; approved requests trigger Calendar Integration and Payroll Adjustment, with an appeal option for rejections" className="block dark:hidden" />
  <img src="/images/uc-human-resources-flow-3-dark.svg" alt="Flowchart of a leave request process: Leave Request Submission, Leave Type Verification (vacation, sick or special), Available Balance Check with medical documentation and compliance review for sick or special leave, then Manager Approval; approved requests trigger Calendar Integration and Payroll Adjustment, with an appeal option for rejections" className="hidden dark:block" />
</Frame>

### Key Process Details

| Stage               | Description                      | Duration | Key Actors     |
| ------------------- | -------------------------------- | -------- | -------------- |
| Request Submission  | Initial leave application        | 30 mins  | Employee       |
| Verification        | Leave type and eligibility check | 1-2 days | HR System      |
| Approval Process    | Manager review and decision      | 2-3 days | Direct Manager |
| Payroll Integration | Leave impact calculation         | 1 week   | Payroll Team   |

## Business Value Proposition

These use cases demonstrate how process automation can:

- Reduce administrative HR overhead by 50-70%
- Improve employee experience and engagement
- Enhance accuracy of HR documentation
- Provide real-time visibility into HR processes
- Standardize and streamline HR workflows
- Minimize human error in critical HR functions
- Enable data-driven decision making
- Ensure compliance with labor regulations
- Accelerate key HR processes
- Improve overall organizational efficiency
