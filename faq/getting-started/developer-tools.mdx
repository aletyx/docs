---
title: "What developer tools are available?"
description: "Aletyx developer tools include a VS Code extension, an online playground, design environments, and Maven and container-registry integration."
---

Aletyx ships a VS Code extension, an online playground, and design environments
(for example, `start.aletyx.ai` and `playground.aletyx.ai`). It also integrates
with Maven and container registries for build and deployment.
