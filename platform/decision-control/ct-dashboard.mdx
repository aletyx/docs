The Aletyx Decision Control Tower dashboard provides at-a-glance visibility into your decision management operations across all environments.

## Dashboard Overview

When you log in to Aletyx Decision Control Tower, you'll see the main dashboard with key metrics and recent activity:

### Key Metrics

- **Total Models**: Number of decision models across all environments
- **Pending Tasks**: Approval tasks awaiting your action
- **Recent Activity**: Actions taken in the last 24 hours
- **Environment Health**: Status of each environment

### Recent Deployments

The dashboard shows the most recent deployment activities:

- Model name and version
- Source and target environment
- Timestamp and status
- User who performed the action

## Navigation Structure

Aletyx Decision Control Tower uses a sidebar navigation with the following views:

### Dashboard

The home view with summary metrics and recent activity.

### Environments

View and manage all Decision Control environments:

- Launch individual environments
- View environment health status
- See model counts per environment
- Access environment URLs

### Units

Browse decision units across environments:

- View all published units
- See versions for each unit
- Compare units across environments
- Access unit details

### Governance

Manage governance workflows:

- View active workflows
- Track approval progress
- See workflow history
- Monitor compliance status

### Tasks

Work with approval tasks:

- View tasks assigned to your role
- Approve or reject requests
- Add comments and justification
- Track task history

### API Explorer

Test and explore Decision Control APIs:

- Execute API calls
- View request/response
- Test with different environments
- Generate code samples

## User Information

The sidebar displays your user information:

- **Username**: Your authenticated identity
- **Role**: Your assigned Decision Control role

Your role determines which environments and features you can access.

## Responsive Design

Aletyx Decision Control Tower adapts to different screen sizes:

### Desktop View

Full sidebar with labels and icons. All features accessible.

### Tablet View

Condensed sidebar with icons. Full functionality maintained.

### Mobile View

Collapsible sidebar. Optimized for touch interaction.

## Session Management

Aletyx Decision Control Tower manages your session through:

- **OAuth2 Tokens**: Secure authentication via Keycloak
- **Auto-Refresh**: Tokens refreshed before expiration
- **Logout**: Clear session and redirect to login

## Notifications

Stay informed about important events:

- New tasks assigned to your role
- Workflow state changes
- Model deployment completions
- System alerts

## Quick Actions

Common actions available from the dashboard:

- **Launch Environment**: Open Decision Control in a new tab
- **View Tasks**: Jump to your pending approvals
- **Recent Models**: Access recently modified models
- **Refresh**: Update dashboard data
