## Start Here

<Tabs>
  <Tab title="Decision Analysts">
    ### For Decision Analysts: Start Modeling in Minutes

    **Visual Decision Modeling with DMN**

    Create business decisions using industry-standard Decision Model and Notation (DMN) without writing a single line of code.

    - Drag-and-drop decision table creation
    - Visual decision requirements diagrams
    - FEEL expression language for business-friendly logic
    - Testing and simulation capabilities

    [Start with DMN](/get-started/playground-dmn)

    **Process Automation with BPMN**

    Design executable business processes that orchestrate people, systems, and decisions.

    - Intuitive BPMN 2.0 visual editor
    - Human task management
    - System integration capabilities
    - Process monitoring dashboard

    [Start with BPMN](/get-started/playground-bpmn)

    **Quick Start** — Get up and running in less than 5 minutes:

    1. Access [Aletyx Playground](https://playground.aletyx.ai) — no installation required
    2. Create a new project or use a starter template
    3. Model your first decision or process
    4. Test with sample data
    5. Export or deploy your solution

    [Open Aletyx Playground](https://playground.aletyx.ai)
  </Tab>
