This path is for **decision analysts** who model business decisions and processes
without writing code, using the visual [Aletyx Playground](https://playground.aletyx.ai).

## Prerequisites

Access to the [Aletyx Playground](https://playground.aletyx.ai) (or a local
deployment via Docker). No development tools required.

## Model a decision with DMN

<Steps>
  <Step title="Open the Playground">
    Go to [playground.aletyx.ai](https://playground.aletyx.ai) and explore a
    sample model — such as the *Loan Pre-qualification* example — to see decision
    nodes, input nodes, and decision tables.
  </Step>
  <Step title="Create a new project">
    Start a blank DMN 1.5 model and give it a name.
  </Step>
  <Step title="Design the decision structure">
    Add input nodes and decision nodes, then set each node's data type (for
    example, `number`).
  </Step>
  <Step title="Connect requirements">
    Draw information-requirement arrows from inputs to the decisions that depend
    on them.
  </Step>
  <Step title="Build decision tables">
    Add conditions and outputs, choose the appropriate hit policy (for example,
    *Collect Max*), and set default values.
  </Step>
  <Step title="Test in real time">
    Use the **Run** feature to validate outputs as you change the model.
  </Step>
  <Step title="Save to Git">
    Connect the project to a Git provider for persistent storage and version
    control.
  </Step>
</Steps>

<Check>
  You've authored, tested, and versioned a decision visually. Next, orchestrate
  it inside a process with BPMN.
</Check>
